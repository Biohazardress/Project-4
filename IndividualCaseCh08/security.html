Web Security

You can access the web in multiple forms through your tablet, laptop, phone, etc. It is important to know what type of data is collected when accessing the internet, no matter how mundane the reason.


Data Collected

  • onLine: This property collects data regarding if your browser has a network connection.

  • platform: This property collects data regarding what operating system you are using.

  • appVersion: This property collects data regarding what web browser version is displaying the site.

  • appName: This property collects data regarding the name of the web browser that is displaying the page.

Resources

Safewise
For CyberSecurity Awareness Month, Safewise compiled a list for parents and kids to use to be safe on the internet. The information provided spans from top threats for kids to what you can do about it.

GFC Global
GFC Global's site provides an introduction to internet safety. The site walks you through the risks of browsing the internet and how to protect yourself from them.

Internet Safety 101
Internet Safety 101 is a safety resource created by Enough Is Enough (EIE). EIE has been the nation's leading internet safety organization since 1994 and has provided multiple resources to protect kids and oneself on the web. (Online Gaming, Social Media, etc.)
